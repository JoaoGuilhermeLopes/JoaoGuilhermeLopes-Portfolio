Projects

Flavoured Water Dispenser

One of my most impactful projects was the development of an IoT-enabled flavoured water dispensing machine, designed to deliver a highly customizable and user-friendly hydration experience.
The system integrated multiple embedded components, including Bluetooth connectivity, EEPROM, real-time pressure/temperature monitoring, dynamic control of pumps/valves and much more.
An intuitive display interface allowed users to adjust settings such as flavour intensity and temperature with precision, while backend systems tracked usage data for maintenance and analytics.
This project exemplified the intersection of hardware and software, requiring seamless coordination between embedded firmware, sensors, actuators, and user interfaces. This has strengthened my skills in embedded systems and communication protocols aswell as reinforcing the importance of user-centric design and robust system reliability in connected products.

Bluetooth Integration

A group of projects involved the integration of Bluetooth Low Energy connectivity into units.
This has enabled users to control and dispense directly from a mobile app, improving convenience and accessibility and enhancing hygiene by minimizing physical interaction with the unit.
In parallel, developed an automated waste kit detection system that allowed the main control boards to determine when the waste container was full—without any physical connection. The system was designed to pair only with the nearest control board, ensuring accuracy and eliminating interference, ultimately contributing to a smarter, more efficient ecosystem.

Automated Testing System

Another key project was the implementation of an automated system for testing units directly on the production line.
This system was designed to automatically validate the core functionalities of each unit—such as sensors, EEPROM, communication modules and control outputs—before they left the factory. By running diagnostic routines, detecting faults in real time, and generating instant feedback, the system reduced the need for manual testing, accelerated production throughput, and improved product quality and consistency.
This initiative greatly streamlined operations while ensuring every deployed unit met standards.

Software Flashing/Traceability Database System

Built around a Raspberry Pi using Python, SQLite, and a custom API, this system interfaced with various microcontrollers (MCUs), enabling automated and flexible firmware flashing across different product lines.
The integrated API allowed seamless communication between the flashing tool and external systems, ensuring real-time data exchange and configurability. Each flashing event—along with key metadata like timestamp, firmware version, unit ID, and operator—was logged into a local database, ensuring traceability. The intuitive user interface streamlined technician workflows, enhanced production efficiency, and improved long-term diagnostic capabilities.

PCB Designs

Design of custom PCBs across a diverse range of products and applications, including embedded systems, sensor integration, wired/wireless communication modules, and power management solutions.
From prototyping to production-ready layouts, design, validation, and testing circuits for mechanical, electrical and environmental requirements. Each project required cross-functional collaboration, adherence to regulatory constraints and understanding of hardware-software co-design—resulting in robust, scalable, and high-performance electronics ready for real-world deployment.
